-- EN FRANÇAIS --

Titre : ZapAPP, les commandes slash et l'IA !

Description : La navigation dans les rencontres de l'association.

ZapAPP est maintenant utilisé par tout le monde dans l'association.

Comment naviguer dans les rencontres MIP ou PJC ?

Pour naviguer dans les rencontres, vous deviez au choix :

  • Vous rendre dans le salon contenant les jams que vous vouliez consulter et chercher votre jam en scrollant ... (Pas très user friendly !)
  • Utiliser le calendrier sur le Blog (là pour MIP ou là pour PJC). Ce qui est déjà mieux mais cela nécessite de sortir de Discord ...

Les commandes slash à la rescousse !

Une commande slash est une commande qui commence par un slash. Dès que vous tapez slash dans la zone de saisie d'un message et que vous avez les droits pour utiliser des commandes, Discord vous propose une liste de choix en autocompletion !

ZapAPP offre maintenant quelques commandes dont vous jugerez vous même la pertinence dans votre façon de naviguer dans vos rencontres Zikapanam:

  • /lister-mes-rencontres : liste les rencontres de l'utilisateur qui a envoyé la commande (sur MIP ou PJC).
  • /prochaines-rencontres : liste les prochaines rencontres liées au serveur Discord sur lequel la commande est lancée (c'est-à dire MIP ou PJC).
  • /prochaines-rencontres-collectifs : liste les prochaines rencontres dans les collectifs dans lesquels vous êtes inscrits (sur MIP ou PJC)
  • /trouver-rencontres : permet de lister toutes les rencontres qui cherchent un instrument spécifique (sur MIP ou PJC). Optionnellement on peut lui dire de ne pas inclure les rencontres que vous organisez ou auxquelles vous participez déjà.
  • /affiche-rencontre : permet d'afficher un résumé synthétique d'une rencontre à partir de son numéro !

Les commandes slash ont des limitations car la taille d'un message Discord est limitée à 2000 caractères, les listes ainsi affichées doivent donc faire moins de 2000 caractères. Très souvent les listes affichées actuellement ne sont pas complètes. Les listes fournissent des liens internet pour chaque rencontre affichée permettant ainsi d'aller directement sur le message original.

L'IA de Zikapanam (ChatGPT)

Vous avez certainement entendu parlé de l'IA ChatGPT ? C'est assez incroyable son fonctionnement et je l'ai câblé sur les Discords de Zikapanam dans des salons privés. Un nombre restreint de membres ZAP y ont eu accès. Je parle dans le passé car le crédit gratuit fournit par ChatGPT s'est envolé très vite !

J'ai entraîné l'IA (ChatGPT) pour qu'elle réponde aux questions sur l'association et les activités proposées. Ça marche plutôt bien en règle générale mais des fois il fabule et invente des choses que je ne lui ai pas dites (Heureusement que c'est pas un médecin ...) !

Néanmoins, je suis plutôt séduit et je pense que l'IA sera un outil supplémentaire pour accueillir les nouveaux arrivants, voir à terme pour répondre à toutes vos questions (Mais pour le moment, techniquement ce n'est pas encore possible d'après mes connaissances).

Cependant, il sera peut-être possible dans quelques temps de lui demander en langage naturel les rencontres à venir ... Sans utiliser directement les commandes slash !

Ceci dit, outre les possibilités que pourrait offrir cette IA, son gros défaut pour nous est son prix ! Il est ramené au nombre de requêtes faîtes à ChatGPT et ça peut vite chiffrer ...

Title: ZapAPP, slash commands and AI!

Description : Navigating association meetings in Discord§.

ZapAPP is now used by everyone in the association.

How to navigate in MIP or PJC meetings

To navigate in the meetings, you had to either:

  • Go to the lounge containing the jams you wanted to view and search for your jam by scrolling ... (Not very user friendly!)
  • Use the calendar on the Blog (here MIP or there PJC). Which is already better, but it requires getting out of Discord ...

Slash commands to the rescue

A slash command is a command that begins with a slash. As soon as you type slash in the input field of a message and have the rights to use commands, Discord offers you an autocomplete list of choices!

ZapAPP now offers a few commands whose relevance to the way you navigate your Zikapanam encounters you'll judge for yourself:

  • /lister-mes-rencontres: lists the encounters of the user who sent the command (on MIP or PJC).
  • /prochaines-rencontres: lists the next encounters linked to the Discord server on which the command is launched (i.e. MIP or PJC).
  • /prochaines-rencontres-collectifs: lists upcoming meetings in the collectives in which you are registered (on MIP or PJC)
  • /find-meetings: lists all meetings looking for a specific instrument (on MIP or PJC). Optionally, you can tell it not to include encounters you're already organizing or participating in.
  • /affiche-rencontre: allows you to display a summary of a meeting based on its number!

Slash commands have limitations because the size of a Discord message is limited to 2000 characters, so lists displayed in this way must be less than 2000 characters long. Very often the lists currently displayed are not complete. The lists provide web links for each encounter displayed, enabling users to go directly to the original message.

I trained the AI (ChatGPT) to answer questions about the association and the activities on offer. It works pretty well as a rule, but sometimes he fabricates and invents things I haven't told him (Good thing he's not a doctor ...)!

Nonetheless, I'm rather seduced and I think AI will be an additional tool to welcome newcomers, or eventually to answer all your questions (but for the moment, technically, it's not yet possible, as far as I know).
